# How to make the good icons?

The best source of the free icons is: https://www.svgrepo.com. Go there, find what you need. Then, adjust the icon main
color and add some balck border to make the icon look more contrast.

## Game/mod presets

* Bottombar icons size: 112x112.
* Mod's small buttons size: 36x36.

## Getting icons from SvgRepo

Icon settings:
- Timberborn main theme color: #BCA26C
- Timberborn "kind of green" color: #6DA26C
- Timberborn "kind of red" color: #FFA26C
- Icon Size: 112px
- Padding: 20% for the bottom bar icons.
- Trace Width (black): 20%
- Save as 256x256 PNG and downscale to the normal 112x112 bottom bar icon size.
