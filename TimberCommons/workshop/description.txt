[h1]Overview[/h1]
TimberCommons is a collection of small UI and quality-of-life improvements for Timberborn. It helps you read the game state faster, tune water-output buildings more precisely, and get better information from compatible irrigation tower mods.

It also provides building components for other mod makers. The technical details are in the Wiki, while this page focuses on what players get in-game.

[h1]For players[/h1]
Check out the image slideshow above to get a picture of what the mod provides. In short:
[list]
[*] [b]Clearer time values[/b]. Shows supply time, growth time, and recipe duration as days/hours where it is easier to read. For example, '2d 4h' is clearer than '52 hours'.
[*] [b]More precise small durations[/b]. Some values in Timberborn are rounded in the stock UI. TimberCommons can show short recipe durations more accurately, such as the Grilled Potatoes recipe taking '0.52h'.
[*] [b]Injury probability[/b]. Buildings that can injure workers can show the injury chance in the right panel or as a small avatar indicator with a tooltip. This is configurable in the settings.
[*] [b]Active mod filter[/b]. Adds a 'Show active' filter to the mod selection dialog, so it is easier to review which installed mods are actually enabled.
[*] [b]Water output control[/b]. Lets you set the output water level for Mechanical Pumps, Deep Mechanical Pumps, Compact Mechanical Pumps, and Discharges. This helps tune reservoirs and avoid accidental flooding.
[*] [b]Better irrigation tower information[/b]. If you install a mod that uses TimberCommons, such as [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3345889146]Water Extension[/url], irrigation tower buildings can show useful details like efficiency, maximum range, growth effects, contamination blocking, and recipe outputs directly in the game UI.
[/list]
