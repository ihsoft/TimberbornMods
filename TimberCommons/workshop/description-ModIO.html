Overview

TimberCommons is a collection of small UI and quality-of-life improvements for Timberborn. It helps you read the game state faster, tune water-output buildings more precisely, and get better information from compatible irrigation tower mods.

It also provides building components for other mod makers. The technical details are in the Wiki, while this page focuses on what players get in-game.

For players

Check out the image slideshow above to get a picture of what the mod provides. In short:

  • Clearer time values. Shows supply time, growth time, and recipe duration as days/hours where it is easier to read. For example, "2d 4h" is clearer than "52 hours".
  • More precise small durations. Some values in Timberborn are rounded in the stock UI. TimberCommons can show short recipe durations more accurately, such as the Grilled Potatoes recipe taking "0.52h".
  • Injury probability. Buildings that can injure workers can show the injury chance in the right panel or as a small avatar indicator with a tooltip. This is configurable in the settings.
  • Active mod filter. Adds a "Show active" filter to the mod selection dialog, so it is easier to review which installed mods are actually enabled.
  • Water output control. Lets you set the output water level for Mechanical Pumps, Deep Mechanical Pumps, Compact Mechanical Pumps, and Discharges. This helps tune reservoirs and avoid accidental flooding.
  • Better irrigation tower information. If you install a mod that uses TimberCommons, such as Water Extension, irrigation tower buildings can show useful details like efficiency, maximum range, growth effects, contamination blocking, and recipe outputs directly in the game UI.

For mod makers

TimberCommons includes reusable irrigation tower components for mods that add new water or badwater-related buildings. See the Wiki for component details, specs, and compatibility notes:
