Style `TimberDevStyle.uss` from this folder should be distributed with the mod asset bundle.
1. Copy the style to the mods assets folder under "UI/Views".
2. Make visual documents, using these style.

Styles:
- "timberdev-minmax-slider" - makes Timberborn theme for MinMaxSlider.
- "timberdev-button-small" - shapes a small button for the entity panel.
