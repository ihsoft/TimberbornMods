Shared storage across a district boundary

Let two adjacent districts use the same warehouse or tank directly.

Each building has two opposite entrances and one synchronized inventory. Beavers from either district can deliver to or collect from the same stock, removing the need for a separate transfer route and extra workers just to move those goods across the boundary.

Included buildings

  • Dual-District Warehouse — stores up to 200 units of boxed goods.
  • Dual-District Tank — stores up to 300 units of liquid.

How to use

Place the building across a district boundary, with one entrance connected to each district's path network. Select what it should store, then use it like normal storage from either side.

Two doors. One stockpile. No paperwork.
