[h1]Timberborn: Custom Tools[/h1] This mod offers new quality-of-life tools in the bottom bar and also provides an API for modders to quickly create their own tools. [hr][/hr] [h2]For the players: Built-in tools[/h2] [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/GroupButtonDemo.png[/img] [h3]Immediate finish of incomplete buildings[/h3] This tool is only visible when the dev mode is activated ([b]Shift + Alt + Z[/b]). Select multiple incomplete buildings and have them [b]instantly completed[/b]. A handy tool when testing or prototyping. [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/FinishNowToolDemo.png[/img] [h3]Pause buildings in the selected range[/h3] Select multiple buildings that need to be paused. Hold [b]SHIFT[/b] to lock the selection to a specific building type. A useful tool for temporarily stopping groups of buildings during colony micromanagement. [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/PauseBuildingToolDemo.png[/img] [h3]Resume buildings in the selected range[/h3] Select multiple buildings you want to resume. Hold [b]SHIFT[/b] to lock the selection to a specific building type. Helps quickly bring groups of buildings back online during micromanagement. [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/ResumeBuildingToolDemo.png[/img] [h3]Adjustable platform[/h3] This tool allows placing 1-, 2-, and 3-levels platforms. Hold [b]ALT[/b] or [b]SHIFT[/b] key to switch between the levels. A convenient tool for building complex platform structures. Additionally, you get [b]CTRL+Z[/b] keybinding which undoes the last platform placement action. [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/AdjustablePlatformToolDemo.png[/img] [h3]Levee or Dam[/h3] This tool allows placing levees or dams. Hold [b]SHIFT[/b] to switch between the two. A convenient tool for building complex water management structures. Additionally, you get [b]CTRL+Z[/b] keybinding which undoes the last levee/dam placement action. [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/LeveeOrDamToolDemo.png[/img] [h3]Path+[/h3] This tool allows placing regular paths, but you also get the [b]CTRL+Z[/b] keybinding which undoes the last path placement action. No need to switch tools if a wrong path was placed! [img]https://raw.githubusercontent.com/ihsoft/TimberbornMods/refs/heads/main/CustomTools/Workshop/Showcase/PathPlusToolDemo.png[/img] [h3]Key bindings[/h3] For every tool you can assign a hotkey. Go to the standard keybindings setup screen in the game. There will be a section for CustomTool bindings. [hr][/hr] [h2]For the modders: API to create your own tools[/h2] For the full instructions, read [url=https://github.com/ihsoft/TimberbornMods/blob/main/CustomTools/README.md]README[/url]. In nutshell, to create your own tool you need: [olist] [*] Implement a class that handles the tool's logic. Use the provided base classes to avoid dealing with the boilerplate. [*] Make a blueprint spec for your tool. [*] [i]Optional[/i]. Create blueprint specs to have your own tool group buttons created. Or you can attach your tool to the CustomTool's group. [*] DONE!
