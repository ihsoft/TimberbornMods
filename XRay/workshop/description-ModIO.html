See what is hidden underground

No need to change the terrain layer now! Just press a hotkey or click the tool button.

✓ See underground structures
✓ Build underground
✓ Select and manage hidden buildings
✓ Multiple visualization modes
✓ Extensive customization options
✓ Toggle or Hold-to-X-Ray

Default Hotkeys

Ctrl+P — Toggle X-Ray
P — Hold-to-X-Ray

All keybindings can be changed in the game settings.
